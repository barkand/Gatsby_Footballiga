# Footballiga Site:

A web3 game on Ethereum blockchain platform along with NFT of Iranian Premier League players.

#### Demo:

> [Demo](footballiga.com/)

#### How we created:

> npm init gatsby site -y -js

#### How to run:

> cd site

> npm i
> npm run start
